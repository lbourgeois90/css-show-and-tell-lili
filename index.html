Text Effect: Text With Shadow

To create text that has a shadow, you first use the css code, color: to select the base color of the shadow. Next you use text-shadow which allows the user to to input values that change the vertical shadow size, horizontal shadow size and set the shadow color. Below are various examples of shadow-text with varying values for those inputs.

Text-Shadow increasing vertical shadow pixels

As you can see, adjusting the vertical shadow value to a larger pixel value will cause the shadow to move below the original text.

Text-Shadow increasing horizontal shadow pixels

As you can see, adjusting the horizontal shadow value to a larger pixel value will cause the shadow to move to the right of the original text.

Text-Shadow decreasing vertical shadow pixels to a negative value

As you can see, adjusting the vertical shadow value to a negative pixel value will cause the shadow to move above the original text.

Text-Shadow decreasing horizontal shadow pixels to a negative value

As you can see, adjusting the horizontal shadow value to a larger pixel value will cause the shadow to move to the left of the original text.
